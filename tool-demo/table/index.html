table shortcode

Use the table shortcode to add color accents and captions to tables. The color affects only the header bottom border — a subtle accent line between headers and data rows.

Syntax

{{< table [color] [caption] >}}
| Col 1 | Col 2 |
|-------|-------|
| a     | b     |
{{< /table >}}

Color

  • (none) → default
  • info
  • note
  • warning
  • success
  • error
  • important

Caption

  • any text string
  • shown above the table
  • optional
