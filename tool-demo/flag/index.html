flag shortcode

Flags are similar to hints but with different styling; a hint is subtle, where a flag demands attention.

Syntax

{{< flag [option] [tint] >}}

Option

  • info → (default)
  • note
  • warning
  • success
  • error
  • important

Tint

  • true
  • false → (default)

Examples

{{% flag info %}}
flags are great for sharing important information. No tint.
{{% /flag %}}
