color shortcode

The color shortcode applies color to inline text without breaking the flow of a sentence — useful when a full hint or flag would be too heavy.

Syntax

{{< color "type" />}}text{{</ /color >}}

Colors match the hint/flag type vocabulary:

  • info → cyan
  • note → purple
  • warning → orange
  • success → green
  • error → red
  • important → yellow

Longer Phrase

This entire sentence is colored warning orange without needing a full hint or flag box. Color can span as much text as needed — words, sentences, or whole paragraphs.
