shortcode basics

Hugo shortcodes are reusable content snippets embedded in markdown. Two delimiter styles control how inner content is processed.

Delimiters

[angle brackets] — raw

Inner content passed through as-is. Use when the cell contains:

  • Other shortcodes
  • Raw HTML
  • No markdown needed

[percent signs] — markdown

Inner content processed as markdown first. Use when the cell contains:

  • Bold, italic text
  • Bullet lists
  • Headings
  • Inline code

Examples

Shortcode inside a shortcode → use {{< >}}

{{< layout cols="1 1" >}}
{{< cell >}}
{{< img src="photo.jpg" >}}
{{< /cell >}}
{{< /layout >}}

Markdown content → use {{% %}}

{{% hint info %}}
This text uses bold and a link.
{{% /hint %}}
info
This text uses bold and a link.

Markdown list in a layout cell → use md shortcode

{{< layout cols="1 1" >}}
{{< md >}}
- Item one
- Item two
- Item three
{{< /md >}}
{{< cell >}}
{{< img src="photo.jpg" >}}
{{< /cell >}}
{{< /layout >}}

Quick Reference

  • angle → {{< >}}
  • percent → {{% %}}

When to use each delimiter

Situation Use
Shortcode inside shortcode angle
Raw HTML inside shortcode angle
Bold, italic, lists inside percent
hint, flag, details content percent
layout, cell, tabgroup container angle
md shortcode content angle

The md shortcode

The md shortcode exists specifically to allow markdown inside {{< >}} contexts like layout. It applies markdownify internally so you get markdown processing without switching delimiter styles.

{{< layout cols="1 1" >}}
{{< md >}}
- markdown list works here
- bold works too
{{< /md >}}
{{< cell >}}
{{< img src="photo.jpg" >}}
{{< /cell >}}
{{< /layout >}}
