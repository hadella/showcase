tab/tabgroup shortcode

The tab/tabgroup shortcodes organize related content into switchable panels — useful for showing the same concept in multiple languages, platforms, or formats.

Syntax

{{< tabgroup id="optional-id" />}}
{{</ tab title="Label" />}}
Content here.
{{</ /tab />}}
{{</ /tabgroup >}}

tabgroup

  • id → unique id for multiple groups on one page (optional)

tab

  • title → tab button label (default: “Tab”)

Code in Tabs

{{< tabgroup />}}
{{</ tab title="Python" />}}
...
{{</ /tab />}}
{{</ /tabgroup >}}

def hello():
    print("Hello, world!")

function hello() {
  console.log("Hello, world!");
}

package main

import "fmt"

func main() {
    fmt.Println("Hello, world!")
}

fn main() {
    println!("Hello, world!");
}

Multiple Tab Groups

Use id to distinguish groups when more than one appears on a page.
