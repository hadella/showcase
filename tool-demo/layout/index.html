layout/cell shortcode

The layout shortcode breaks out of markdown’s single-column flow. Use cell for shortcode/HTML content, md for markdown content. Both can be mixed within the same layout.

Syntax

{{< layout [cols] [gap] [align] />}}
{{</ cell />}}...{{</ /cell />}}
{{</ md />}}...{{</ /md />}}
{{</ /layout >}}

layout params

  • cols → fr values e.g. "1 2" (default: equal)
  • gap → CSS value e.g. "1rem" (default: 0)
  • align → top, center, bottom (default: top)

cell / md params

  • text → left, center, right, justify
  • use cell for shortcodes and HTML
  • use md for markdown lists, headings, etc.
