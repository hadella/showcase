The `space` shortcode provides a mechanism to add some extra vertical space between various elements in a post.

{{< hint note >}}
Markdown ignores whitespace between paragraphs
{{< /hint >}}

## Example

Let's say we have a few paragraphs with multiple blank lines in between.

```
Lorem ipsum ...



Lorem ipsum ...
```

{{% hint warning %}}
Markdown will **ignore** those blank lines in between. So the rendered text looks like:
{{% /hint %}}

{{% hint info %}}
However, we can use the **space** shortcode to add vertical space in between.
{{% /hint %}}

```
Lorem ipsum ...

{{</* space "3rem" */>}}

Lorem ipsum ...
```
